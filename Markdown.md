#Markdown基本语法笔记

#####1、标题
```
# 一级标题
## 二级标题  
### 三级标题  
#### 四级标题  
##### 五级标题  
###### 六级标题   
```

#####2、段落标记
2个空格或者直接换行  

#####3、字体
```
*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___
```

#####4、列表
```
* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项


- 第一项
- 第二项
- 第三项
```
#####5、区块
```
> 区块引用
> 菜鸟教程
> 学的不仅是技术更是梦想
```
#####6、代码
反引号（数字1旁边）
`这是代码`

#####7、代码块
```三个反引号```

#####8、链接
`[链接名称](链接地址)`
或者
`<链接地址>`
